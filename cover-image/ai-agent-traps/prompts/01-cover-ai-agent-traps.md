# Final image generation prompt (reproducibility record)

Create a 16:9 conceptual cover image for a Chinese tech article.

Title (must be clearly readable, Chinese):
(译文)AI Agent 陷阱：了解你的 AI Agent 是如何被毒害的

Subtitle (smaller, English):
AI Agent Traps: Poisoning the Agent's Information Environment

Design goal:
Visualize that the attack surface is the information environment around AI agents, not just model internals.

Composition (three-zone horizontal structure):
1) Left zone: untrusted information inputs (abstract icons for web pages, email, documents, API responses), flowing inward.
2) Center zone: an AI agent interpretation core shown as layered nodes (perception, reasoning, memory, action) in abstract, non-literal form.
3) Right zone: outcomes and risks (decision distortion, data exfiltration, systemic instability) using iconographic symbols like warning triangle, broken shield, leak arrows, without sensational gore.

Style and look:
- conceptual + digital editorial style
- polished, precise edges, subtle gradients, clean card/grid layout
- technical, professional, calm
- lots of breathing room (40-50% whitespace)
- no photorealistic humans

Color direction (for rendering guidance only):
- primary technical blue family (engineering blue, navy, cyan)
- light gray / off-white background
- tiny amber accents for risk highlights
- low contrast, subtle mood

Typography:
- clean modern sans-serif
- Chinese title prominent and crisp
- English subtitle secondary but legible

Strict constraints:
- Do NOT show color names, hex values, palette labels, or watermark text
- Do NOT include logos, brand marks, URL, or extra unrelated text
- Do NOT make it look like a horror poster
